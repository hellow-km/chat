# 一个可以简易的加好友聊天的项目，目前不能实时刷新，websocket可以做，以后再做

### 装依赖
```
npm install
```

### 运行
```
npm run serve
```

### 运行gui
```
npm run dev
```

### 打包
```
npm run build
```

### 启动后端
```
cd server/wechat
node app
```
